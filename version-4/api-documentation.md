## Country Counts

---

### ðŸ”¹ POST `/update-one-country-count`

**Description:** Increments the view count of a given country. If it doesn't exist, initializes it with count 1. Returns a response with the updated country's count. 

**Request Body:**

```json
{
  "country_name": "France"
}
```

**Response:**

```json
{
  "count": 3
}
```
